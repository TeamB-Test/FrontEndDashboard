## Features

- 🍔 **Food Management** — full CRUD with image upload (multipart/form-data), sorting, filtering, and pagination
- 📁 **Category Management** — create, edit, and delete food categories with an interactive UI
- 📦 **Order Management** — filter orders by Pending / Confirmed / Delivered, update status via API
- 📊 **Real-time Dashboard** — live order counts, revenue totals, and best-selling items using WebSockets and polling
- 🐛 **Bug-tracked workflow** — all issues managed via Jira, resolved through GitHub PRs

## Getting Started

```bash
# Clone the repo
git clone https://github.com/yourusername/tabletapster.git

# Install dependencies
cd tabletapster
npm install

# Start the development server
npm run dev
```

> Make sure the backend API is running and update the base URL in your Axios config.
